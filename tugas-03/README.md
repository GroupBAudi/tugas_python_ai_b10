# Tugas 3 - Python Basics

Repository ini dibuat untuk memenuhi tugas Python Basics.

## Isi Repository

- `tugas3.py` → file Python utama untuk tugas

## Fitur Program

Program di dalam `tugas3.py` mencakup:

- Deklarasi variabel dengan beberapa tipe data
- Manipulasi string
- Operasi matematika sederhana
- Penggunaan list dan akses elemen
- Input nama dan umur dari user
- Output perkenalan sederhana

## Cara Menjalankan

1. Pastikan Python sudah terinstall
2. Download atau clone repository ini
3. Jalankan file berikut:

```bash
python tugas3.py
```

> Note: Linux gunakan `python3 tugas3.py` instead.

## Output Program

Program akan menampilkan:

- contoh variabel dan tipe data
- hasil manipulasi string
- hasil operasi matematika
- pengolahan list
- input nama dan umur dari user
