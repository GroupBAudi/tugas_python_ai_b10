## Cara Menjalankan

1. Buat virtual environment dan install dependency:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

2. Masuk ke folder tugas:

```bash
cd tugas-03-06/
```

3. Jalankan program:

```bash
python tugas3.py
python tugas4.py
python tugas5.py
python tugas6.py
```
