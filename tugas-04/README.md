# Tugas 4 - Python Data Structures

Repository ini berisi tugas Python Data Structures.

## File
- `tugas4.py`

## Deskripsi
File `tugas4.py` mencakup:
- List: akses, slicing, append, insert, extend, pop, remove
- Tuple: len, indeks, unpacking
- Set: union, intersection, difference, symmetric difference
- Dictionary: tambah, ubah, hapus, keys, values, items
- Nested structures
- Comprehension
- Keanggotaan dan pencarian sederhana

## Cara Menjalankan

```bash
python tugas4.py
```

> Note: Linux gunakan `python3`
